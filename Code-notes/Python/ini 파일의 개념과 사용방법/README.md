## 📦 주요 내용

- `.ini` 설정 파일을 사용하여 Key-Value 저장 및 읽기
- 존재하지 않는 섹션(`Section`)이나 키(`Key`)일 경우 자동으로 생성
- 기본값을 설정하고, `.ini` 파일이 없을 경우 새로 생성
